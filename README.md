# WCMOON-UI

使用 svelte 开发的 web components 组件库

激情开发中。。。

## 开发

```
yarn 
yarn dev
```

开发环境源代码在 example 文件夹

所有组件集合在 packages 文件夹

编译结果在 lib 文件夹

## 使用

```sveltehtml
// 以 table 组件为例
// script
<script >
  import 'wcmoon-ui/lib/wm-table';
</script>

// html
<wm-table />
```
